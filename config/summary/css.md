# 弹性盒子中 flex: 0 1 auto 表示什么意思？
flex属性是flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto

flex:1 === flex-grow: 1; flex-shrink: 1; flex-basis: 0%;
